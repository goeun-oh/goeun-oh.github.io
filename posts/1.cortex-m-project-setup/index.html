STM32CubeIDE н”„лЎңм қнҠё мғқм„ұ

мқҙлІҲ нҸ¬мҠӨнҠём—җм„ңлҠ” STM32CubeIDEлҘј мӮ¬мҡ©н•ҳм—¬ мғҲ н”„лЎңм қнҠёлҘј мғқм„ұн•ҳкі  кё°ліём Ғмқё м„Өм •мқ„ м§„н–үн•ңлӢӨ.

кё°ліё н”„лЎңм қнҠё м„Өм •

н”„лЎңм қнҠё мғқм„ұ лӢЁкі„

1. STM32CubeIDE мӢӨн–ү

  • STM32CubeIDEлҘј мӢӨн–үн•ңлӢӨ
  • вҖңFile вҶ’ New вҶ’ STM32 ProjectвҖқ м„ нғқ

2. MCU м„ нғқ

  • Target Selectionм—җм„ң STM32F411RE кІҖмғү л°Ҹ м„ нғқ
  • Nucleo-64 ліҙл“ң м„ нғқ

3. н”„лЎңм қнҠё м„Өм •

1. RCC (нҒҙлҹӯ) м„Өм •

нҒҙлҹӯ м„Өм •мқҖ STM32 к°ңл°ңм—җм„ң к°ҖмһҘ мӨ‘мҡ”н•ң мҙҲкё° м„Өм • мӨ‘ н•ҳлӮҳлӢӨ.

1) нҒҙлҹӯ мҶҢмҠӨ мқҙн•ҙ

лӮҙл¶Җ нҒҙлҹӯ (HSI - High Speed Internal)

  • мЈјнҢҢмҲҳ: 16MHz кі м •
  • нҠ№м§•:
    • мҷёл¶Җ нҒ¬лҰ¬мҠӨн„ёмқҙ н•„мҡ” м—ҶлӢӨ
    • мҳЁлҸ„лӮҳ м „м•• ліҖнҷ”м—җ мҳҒн–Ҙмқ„ л°ӣмқ„ мҲҳ мһҲлӢӨ
    • м •нҷ•лҸ„к°Җ мҷёл¶Җ нҒҙлҹӯліҙлӢӨ лӮ®лӢӨ

мҷёл¶Җ нҒҙлҹӯ (HSE - High Speed External)

MCU мҷёл¶Җм—җ мӢӨм ң нҒ¬лҰ¬мҠӨн„ё мҳӨмӢӨл Ҳмқҙн„° л¶Җн’Ҳмқ„ лӢ¬м•„м„ң мӮ¬мҡ©н•ҳлҠ” нҒҙлҹӯмқҙл©° ліҙнҶө ліҙл“ңм—җ нҒ¬лҰ¬мҠӨн„ёмқҙлӮҳ нҒҙлҹӯ лӘЁл“Ҳмқҙ лӮ©л•ңлҗҳм–ҙ мһҲмқҢ

  • мЈјнҢҢмҲҳ: 8MHz (мҡ°лҰ¬ ліҙл“ң кё°мӨҖ)
  • нҠ№м§•:
    • мҷёл¶Җ нҒ¬лҰ¬мҠӨн„ё мҳӨмӢӨл Ҳмқҙн„°лҘј мӮ¬мҡ©н•ңлӢӨ
    • лҶ’мқҖ м •нҷ•лҸ„мҷҖ м•Ҳм •м„ұмқ„ м ңкіөн•ңлӢӨ
    • м •л°Җн•ң нғҖмқҙл°Қмқҙ н•„мҡ”н•ң м• н”ҢлҰ¬мјҖмқҙм…ҳм—җ м Ғн•©н•ҳлӢӨ

2) RCC м„Өм • л°©лІ•

1. Clock Configuration нғӯмңјлЎң мқҙлҸҷ

RCC нҒҙлҰӯ

2. HSE м„Өм •

RCC вҶ’ High Speed Clock (HSE) вҶ’ Crystal/Ceramic Resonator м„ нғқ

3. PLL м„Өм •

  • HSE(8MHz)лҘј PLL нҡҢлЎңлҘј нҶөн•ҙ 100MHzлЎң мҰқнҸӯн•ңлӢӨ
  • мқҙ кіјм •мқ„ нҶөн•ҙ мӢңмҠӨн…ң нҒҙлҹӯмқ„ мөңлҢҖ м„ұлҠҘмңјлЎң м„Өм •н•  мҲҳ мһҲлӢӨ

4. мөңмў… нҒҙлҹӯ м„Өм •

  • HCLK: 100MHz (мӢңмҠӨн…ң нҒҙлҹӯ)
  • APB1: 50MHz (м ҖмҶҚ мЈјліҖмһҘм№ҳ)
  • APB2: 100MHz (кі мҶҚ мЈјліҖмһҘм№ҳ)
  • Timer Clock: 100MHz

мӨ‘мҡ”: PLLмқ„ мӮ¬мҡ©н•ҳл©ҙ м ҖмЈјнҢҢмҲҳ мһ…л Ҙмқ„ кі мЈјнҢҢмҲҳлЎң ліҖнҷҳн•ҳм—¬ мӢңмҠӨн…ң м„ұлҠҘмқ„ мөңлҢҖнҷ”н•  мҲҳ мһҲлӢӨ.

2. SYS (мӢңмҠӨн…ң) м„Өм •

Debug м„Өм •

л””лІ„к№…мқ„ мң„н•ң мӢңмҠӨн…ң м„Өм •мқ„ м§„н–үн•ңлӢӨ.

Debug мқён„°нҺҳмқҙмҠӨ м„ нғқ:

SYS вҶ’ Debug вҶ’ Serial Wire м„ нғқ

Serial Wire Debug (SWD)мқҳ мһҘм җ:

  • н•Җ мҲҳ м Ҳм•Ҫ: JTAG лҢҖл№„ м ҒмқҖ н•Җ мӮ¬мҡ© (SWDIO, SWCLKл§Ң н•„мҡ”)
  • кі мҶҚ л””лІ„к№…: нҡЁмңЁм Ғмқё л””лІ„к№… мқён„°нҺҳмқҙмҠӨ м ңкіө
  • мӢӨмӢңк°„ м¶”м Ғ: мӢӨмӢңк°„мңјлЎң н”„лЎңк·ёлһЁ мӢӨн–ү мғҒнғңлҘј лӘЁлӢҲн„°л§Ғн•  мҲҳ мһҲлӢӨ

м°ёкі : SWDлҠ” ARM Cortex-M мӢңлҰ¬мҰҲм—җм„ң н‘ңмӨҖмңјлЎң мӮ¬мҡ©лҗҳлҠ” л””лІ„к№… мқён„°нҺҳмқҙмҠӨлӢӨ.

3. GPIO м„Өм •

PA5 н•Җмқ„ OutputмңјлЎң м„Өм •

User LD2лҘј м ңм–ҙн•ҳкё° мң„н•ҙ PA5 н•Җмқ„ GPIO OutputмңјлЎң м„Өм •н•ңлӢӨ.

м„Өм • лӢЁкі„:

1. Pinout & Configuration нғӯм—җм„ң PA5 н•Җ м°ҫкё°

2. PA5 нҒҙлҰӯ нӣ„ GPIO_Output м„ нғқ

3. GPIO м„Өм • нҷ•мқё

  • GPIO output level: Low (мҙҲкё°к°’)
  • GPIO mode: Output Push Pull
  • GPIO Pull-up/Pull-down: No pull-up and no pull-down
  • Maximum output speed: Low

GPIO лӘЁл“ң м„ӨлӘ…

Output Type мҳөм…ҳ:

Push Pull

  • кё°ліё м„Өм •мңјлЎң лҢҖл¶Җл¶„мқҳ мҡ©лҸ„м—җ м Ғн•©н•ҳлӢӨ
  • HIGH/LOW м–‘л°©н–ҘмңјлЎң нҷ•мӢӨн•ң мӢ нҳёлҘј м¶ңл Ҙн•ңлӢӨ
  • LED м ңм–ҙм—җ м Ғн•©н•ҳлӢӨ

Open Drain

  • мЈјлЎң I2C нҶөмӢ мқҙлӮҳ мҷҖмқҙм–ҙл“ң OR лЎңм§Ғм—җ мӮ¬мҡ©н•ңлӢӨ
  • HIGH мһ„н”јлҚҳмҠӨ мғҒнғңмҷҖ LOWл§Ң м¶ңл Ҙн•  мҲҳ мһҲлӢӨ

Pull-up/Pull-down мҳөм…ҳ:

  • No pull: мҷёл¶Җ н’Җм—…/н’ҖлӢӨмҡҙ м Җн•ӯмқ„ мӮ¬мҡ©н•  л•Ң
  • Pull-up: лӮҙл¶Җ н’Җм—… м Җн•ӯ мӮ¬мҡ©
  • Pull-down: лӮҙл¶Җ н’ҖлӢӨмҡҙ м Җн•ӯ мӮ¬мҡ©

4. м„Өм • кІ°кіј нҷ•мқё

Clock Tree нҷ•мқё

Clock Configuration нғӯм—җм„ң мөңмў… нҒҙлҹӯ м„Өм •мқ„ нҷ•мқён•  мҲҳ мһҲлӢӨ:

  • Input: HSE 8MHz
  • PLL: x12.5 (8MHz Г— 12.5 = 100MHz)
  • SYSCLK: 100MHz
  • AHB: 100MHz
  • APB1: 50MHz (Г·2)
  • APB2: 100MHz (Г·1)

GPIO м„Өм • нҷ•мқё

Pinout viewм—җм„ң лӢӨмқҢмқ„ нҷ•мқён•  мҲҳ мһҲлӢӨ:

  • PA5к°Җ л…№мғүмңјлЎң н‘ңмӢңлҗңлӢӨ (GPIO_OutputмңјлЎң м„Өм •лҗЁ)
  • н•Җ лқјлІЁмқҙ GPIO_OutputмңјлЎң ліҖкІҪлҗңлӢӨ

к°„лӢЁн•ң н…ҢмҠӨнҠё мҪ”л“ң

мғқм„ұлҗң main.cм—җ LED м ңм–ҙ мҪ”л“ңлҘј м¶”к°Җн•ҙліҙмһҗ:

/* USER CODE BEGIN WHILE */
while (1)
{
    /* USER CODE END WHILE */
    
    // LED ON
    HAL_GPIO_WritePin(GPIOA, GPIO_PIN_5, GPIO_PIN_SET);
    HAL_Delay(500);  // 0.5мҙҲ лҢҖкё°
    
    // LED OFF
    HAL_GPIO_WritePin(GPIOA, GPIO_PIN_5, GPIO_PIN_RESET);
    HAL_Delay(500);  // 0.5мҙҲ лҢҖкё°
    
    /* USER CODE BEGIN 3 */
}
/* USER CODE END 3 */
