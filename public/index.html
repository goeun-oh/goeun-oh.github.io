🚀 주요 프로젝트

📱 개인 포트폴리오 웹사이트

Hexo GitHub Pages Material Design JavaScript

  • 반응형 디자인: 모든 디바이스에서 최적화된 사용자 경험 제공
  • 성능 최적화: 빠른 로딩 속도와 SEO 최적화로 검색 노출 향상

React Node.js MongoDB Express

  • 사용자 인증: 안전한 개인 데이터 관리

Next.js TypeScript PostgreSQL Stripe
